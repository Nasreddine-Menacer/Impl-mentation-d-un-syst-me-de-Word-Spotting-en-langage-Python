Word Spotting



Le repérage de mots-clés (ou plus simplement, le repérage de mots ) est un problème historiquement défini pour la première fois dans le contexte du traitement de la parole . Dans le traitement de la parole, le repérage des mots-clés concerne l'identification des mots - clés dans les énoncés. Le repérage de mots-clés est également défini comme un problème distinct, mais apparenté, dans le contexte du traitement d'images de documents . Dans le traitement d'image de document, le repérage de mots-clés est le problème de trouver toutes les instances d'un mot de requête existant dans une image de document numérisée, sans le reconnaître complètement.
